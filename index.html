Conteúdo Programático

Este curso, tem como objetivo, capacitar o aluno nas principais tecnologias front-end utilizadas no mercado de trabalho. Serão utilizadas boas práticas de desenvolvimento, visando a implementação do código com consistência, escalável, de fácil manutenção e com uma boa semântica. O curso será dividido em 3 fases:

Fase Kuririn

Serão ensinadas, com atividades práticas, as seguintes tecnologias:

  • Introdução
  • HTML5
  • CSS
  • Git
  • Prática 1: Utilizando o Git em uma aplicação HTML-CSS - Estilizando o HTML Criando uma landing page(Startup)
Fase Piccolo

Aqui, através de atividades práticas, os alunos aprenderão:

  • CSS3
  • Bibliotecas CSS e JavaScript
    1. jQuery
    2. Materialize
    3. Bootstrap
  • JavaScript.
  • Prática 2: Transformar a landing page em um site responsivo Mobile-First.
  • Prática 3: Criando meu animal virtual.
  • Prática 4: Validação de Formulários.
Fase Goku

Será ensinado e colocado em prática nesta fase:

  • JavaScript: Orientação a Objetos.
  • Automatizador de tarefas.
  • Processadores de CSS.
  • Prática 5: Carregando conteúdo dinâmicamente.
  • Prática 6: Projeto Geolocalização (HTML5 + CSS3 + JS + Leaflet.js).
