Desenvolvimento Web Z

Uma abordagem Saiyajin

Sobre

O curso Desenvolvimento Web Z foi feito para quem deseja cair de cabeça no mundo do desenvolvimento Web com foco em Front End.

Os alunos que participarem deste curso serão treinados por discípulos do Mestre Kame e receberão a doutrina do HTML, CSS e Javascript através de aulas expositivas e muitas atividades práticas.

Além de acesso a metodologia do Mestre Kame os alunos irão aprender sobre tecnologias importantíssimas no mundo do desenvolvimento Front End como Bootstrap, Materialize, jQuery, LeafletJS, entre outros

Modulos

Objetivando a didática este curso será entregue no formato de 3 módulos sendo eles fundamentados respectivamente em cima de cada um dos pilares do desenvolvimento Front End (HTML, CSS e Javascript).

Módulo Kuririn
Conteúdos:
  • HTML5 e seus elementos
  • Mergulhando no Bootstrap
  • Controlando versões com Git
Módulo Piccolo
Conteúdos:
  • Estilizando com CSS
  • Animando com CSS3
  • Desenvolvendo Mobile-first
Módulo Goku
Conteúdos:
  • JavaScript
  • JavaScript Orientado a Objetos
  • Bibliotecas úteis

Os mestres

Gláuber Brennon

Em 2012, participou de diversos projetos Saffira (Sistema de Segurança Florestal), BSLS (Sistema de Segurança e Sinalização Luminosa voltada para ciclistas), ScanRT, Anson Framework, Kinesis, AutoSec e foi co-fundador Automatronic Wearable Devices. Em 2013, ganhou o segundo lugar no concurso 'Tecnologias que Transformam', na Campus Party. Em 2014, foi desenvolvedor na ADSS - Presença digital. Tem experiência em Linux Embarcado, MySQL, Zend PHP.

Marcos Silva

Em 2009, fez o curso de desenvolvimento Java e buscou aprender outras linguagens e tecnologias tais como: PHP, Html, CSS, Javascript. Em 2010, entrou no curso de BCC na UFBA e desde então fez Iniciação ciêntifica e estágios adiquirindo experiências nas mais variadas áreas, tais como, mineração de dados, visão computacional, visualização de dados, em diversos frameworks, o Yii, o Wordpress, o Drupal, o Flask, em diversas linguagens de programação o C/C++, o Python, o Haskell, o Ruby, entre outras. Atualmente é lider de projeto e desenvolvedor no projeto GIIRO. Neste projeto utiliza HTML5, CSS3, Javascript, no front end, e no back end utiliza a linguagem de programação Python com o micro-framework Flask, o ORM SQLAlchemy com a extensão GeoAlchemy2 e o banco de dados Postgres com a extensão PostGis.

Víctor Romário

Entrou para o mundo de desenvolvimento web em 2014, devido ao projeto de Iniciação Científica que tinha como objetivo integrar a linguagem CUDA com JavaScript. Atualmente estagia como desenvolvedor no projeto GIIRO (Gerenciador de Informações e Infraestrutura de Rede Óptica da Rede Nacional de Ensino e Pesquisa (RNP)), no Ponto de Presença do Estado da Bahia (POP-BA) na UFBA. Neste projeto utiliza HTML5, CSS3, Javascript, no front end, e no back end utiliza a linguagem de programação Python com o micro-framework Flask, o ORM SQLAlchemy com a extensão GeoAlchemy2 e o banco de dados Postgres com a extensão PostGis.
